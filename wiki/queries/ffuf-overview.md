# FFUF (Fuzz Faster U Fool)

FFUF è uno strumento di sicurezza scritto in Go per il web fuzzing.

## Funzionalità
- Directory Discovery
- File Discovery
- Subdomain Enumeration
- Parameter Fuzzing

## Utilizzo in Aichain
1. **Security Audit**: Ricerca di file esposti (es. .env, backup).
2. **Concorrenza**: Analisi architetturale di siti competitor.

*Nota: Da usare con cautela su server di produzione.*
